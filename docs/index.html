VedicDateTime

Vedic calendar (Panchanga) computations for R

VedicDateTime

An open-source R package to compute key entities from the Vedic (lunisolar) calendar and convert Gregorian dates (or Julian Day Numbers) into Panchanga components such as tithi, vaara, nakshatra, yoga, and karana.

CRAN: cran.r-project.org/package=VedicDateTime
GitHub: github.com/neerajdhanraj/VedicDateTime

Package status

What you get

Panchanga entities

Compute:

  • tithi
  • vaara
  • nakshatra
  • yoga
  • karana

Conversions

  • Gregorian → JD (gregorian_to_jd())
  • JD → Gregorian (jd_to_gregorian())

Location-aware results

Many computations depend on:

  • latitude
  • longitude
  • timezone offset

Time-series features

Transform timestamps into Vedic features (tithi/nakshatra/yoga/masa) for modeling pipelines.

Quick start

install.packages("VedicDateTime")

library(VedicDateTime)
place <- c(15.34, 75.13, +5.5)
jd <- gregorian_to_jd(8, 9, 2022)

tithi(jd, place)

Use cases

  • Calendar-aware feature engineering for time series
  • Research workflows requiring lunisolar calendar entities
  • Reproducible calendrical computations for a specified location
